# Contributing to TalentGate

Thank you for your interest in TalentGate.

TalentGate is currently a **founder-led project** under active development. At this stage, the repository is **not open to external code contributions**.

---

## Contribution Policy

- This repository is **view-only** for the public.
- Pull requests from external contributors are **not accepted** unless explicitly requested.
- Forking the repository for the purpose of creating derivative products is **not permitted** under the project license.

However, feedback and discussion are welcome.

## Development Status

TalentGate is in **early-stage development**. APIs, data models, and UI are evolving rapidly and may change without notice.

There is currently:
- No public roadmap for external contributors
- No stable plugin or extension interface
- No compatibility guarantees between versions

---

## Future Contributions

As the project matures, contribution guidelines may be updated to allow:
- Documentation contributions
- Bug fixes
- Feature proposals via RFC-style discussions

Any changes to contribution policy will be reflected in this file.

Thank you for respecting the project boundaries and for your interest in TalentGate.
